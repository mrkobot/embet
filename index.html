Free at the-odds-api.com — 500 requests/month, no card needed.
2 Cloudflare Worker URL OPTIONAL
Without this, the math layer runs alone. With it, every AI provider you configure researches and competes daily to pick the best play.
The Worker holds your AI provider keys and routes requests. You control which models compete by setting their keys.
All three models search the live web before picking — injuries, line moves, weather.
1. Create a free account at cloudflare.com
2. Go to Workers & Pages → Create → Hello World → Deploy
3. Click Edit Code → clear everything → paste the code below → Save and Deploy
4. Go to Settings → Variables. Add any of these environment variables as Secret type, not Text:
ANTHROPIC_KEY — console.anthropic.com
OPENAI_KEY — platform.openai.com
GEMINI_KEY — aistudio.google.com/apikey
5. Copy the Worker URL from the Overview tab and paste it in the field above
Only add keys for providers you have. Each provider key needs its own active billing — a ChatGPT Plus, Gemini Advanced, or Claude Pro subscription does not cover API usage. Live search calls (Claude, GPT-5.5, Gemini) cost slightly more per call than a plain text response — still a few cents a day at this volume. To change model versions, edit MODEL_CONFIG in index.html and push the site — the Worker never needs to change for this.
