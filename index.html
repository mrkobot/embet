Deploy a secure, free API firewall to protect your AI keys in under 5 minutes:

1. Sign Up Free: Create a standard free tier profile at dash.cloudflare.com.

2. Deploy Worker: Go to Workers & Pages → Create Application → Create Worker. Choose "From Hello World" and click Deploy.

3. Edit Script: Click Edit Code. Wipe out all placeholder lines completely, paste your core pipeline script into line 1, and click Save and Deploy.

4. Add Keys: Exit the editor back to the worker dashboard. Go to Settings → Variables → Environment Variables. Click Add variable and add entries using these exact names (All Caps):

  • `ANTHROPIC_KEY`: Your Claude Token
  • `OPENAI_KEY`: Your OpenAI Token
  • `GEMINI_KEY`: Your Google Gemini Token
  • `DEEPSEEK_KEY`: Your DeepSeek Token

Only add lines for models you actually want to use. Leave the rest blank!

5. Fetch Live URL: Head back to your worker's Overview tab. Copy the web address under Live URL on the right side. Paste that text block into Slot 2 above.
